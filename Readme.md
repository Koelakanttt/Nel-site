## О проекте

Корпоративный сайт для IT-студии, занимающейся веб-разработкой, SEO-продвижением и IT-консалтингом (сегмент B2B). Проект решает три задачи:

- Автоматизация сбора требований от клиентов через интерактивный квиз
- Динамичное портфолио с фильтрацией без перезагрузки страницы
- Удобная панель управления контентом — без разработчика
---

## Стек технологий

| Слой | Технологии |
|---|---|
| Frontend | HTML5, CSS3 (Flexbox, Grid), JavaScript, Fetch API |
| Backend | PHP, REST-подобный API (JSON) |
| База данных | MySQL, 9 таблиц, 3НФ |
| Авторизация | OAuth 2.0 (Яндекс ID) |
| Редактор | TinyMCE |
| Сервер | Apache, Linux |

---

## База данных

Физическая модель включает 9 взаимосвязанных таблиц. Каждой таблице назначен суррогатный первичный ключ (id, autoincrement). Связи реализованы через внешние ключи — целостность данных на уровне СУБД. База нормализована до третьей нормальной формы.
